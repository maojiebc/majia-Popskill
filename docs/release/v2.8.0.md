# Popskill v2.8.0 — 成熟度大版

按「成熟 Mac app」标准做的一轮系统性体检与修复：8 维度多 agent 审计出 41 项已确认缺陷，本版全部修完（本地化除外，单独立项）。没有新功能面，全部是把已有承诺做实。

## 数据安全

- **回收站按入站时间轮换**：旧实现按内容 mtime 排序，半年没改过的技能刚备份进回收站就可能被当「最旧」物理删除；上限 20 → 200 份，一次大套装更新不再冲爆全局额度。
- **更新原子化**：换版先拷到隐藏临时名，拷贝失败（磁盘满）旧版无损；换名失败自动回滚。旧实现先弃旧版再拷，失败即丢数据 + symlink 全断。
- **回收站终于有 UI**：设置新增回收站区块——列最近条目、一键恢复到 store、打开文件夹。全 app 三处「可恢复」承诺首次兑现。

## 诚实的反馈

- 断网时检查更新不再谎报「全部源已是最新」——失败和最新分开计数，手动检查如实报告。
- 「全部修复」按实际成功数汇报，失败原因不再被成功提示覆盖。
- 错误提示红色、停留 6 秒；关键操作全部留 os.Logger 日志，Console.app 按 com.majia.popskill 可回看。
- 修复弹层「从源重新拉取」移到后台执行——曾在主线程同步 git clone，网络慢时整个界面冻死。

## 性能

- **检查更新快一个量级**：先 `git ls-remote` 比对上游 HEAD，commit 没动就不整仓 clone——多源启动检查从分钟级降到秒级。
- 卡片网格懒加载，搜索逐键不再全量重建屏外卡片。
- git 子进程加超时看门狗 + 管道防死锁，网络黑洞不再挂死。

## 平台惯例

- ⌘, 打开设置、⌘R 手动刷新、⌘F 聚焦搜索、Help 菜单链到文档。
- app 回到前台自动重扫——在终端改了 ~/.agents 不用再重启。
- 单窗口化（去掉会复制界面的 ⌘N）；深色模式下系统弹窗与主窗外观统一。
- VoiceOver 全面适配：状态格念「能力 · 工具：已激活/断链」而非字形名；对比度按 WCAG 加深三档文字色。

## 其他

- 安装 GitHub 仓库的 store 目录名修正为仓库名（曾是随机临时名）。
- npm 来源的过期承诺全部清理；类型过滤空结果的病句修正。
- Sparkle 自动检查可在设置里关闭；新增「报告问题…」一键预填 GitHub issue。
- 发版链一键化（scripts/release.sh + VERSION 单源 + build 号断言），本地 CI 全链重写。
