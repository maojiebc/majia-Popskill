# Popskill v2.4.1 — 新用户引导打通

## 修复 / 新增

- **空态「扫描本地目录」真正可用**：此前只重扫空 store（对新用户是哑的）。现在 store 为空时会扫描 `~/.claude` / `~/.codex` 里的存量技能目录，确认后一键收编进 store 并原位换 symlink（原目录进回收站可恢复）——典型 Claude Code 老用户装完 app 点一下，矩阵直接成型。
- **未安装的工具默认不挂载**：添加能力时，不存在的工具根（如没装 Codex）默认关闭开关，不再凭空创建目录。
- 微小技能不再显示「0.0k tokens」。

新用户三幕实测通过：全空机器 → 空态引导；存量 Claude 用户 → 扫描收编成矩阵；零基建首次安装 → 自建 store。测试 41/41。
