# Popskill v2.1.2 — 实机反馈打磨

首批真机使用反馈的集中修复版。

## 修复

- **排序**：套装永远置顶（按名称），独立能力按 类型（Skill→Agent→MCP→CLI）→ 名称 排列——不再是目录扫描的字母序
- **类型识别**：`guancli` 这类 CLI 包装技能不再误标为 SKILL——frontmatter 显式 `type:` 优先，名称特征（`-cli` / `-mcp`）兜底；展示类型与链接布局解耦，symlink 仍写在 `skills/` 不受影响
- **套装收编**：既不在 lock 文件、frontmatter 又没 homepage 的同族散件（如 `baoyu-diagram`），按前缀族启发并入源式套装（≥5 个成员共享前缀才触发，更新检查时会按上游实际目录核实）
- **应用图标**：内容缩入 Apple 图标网格（824/1024 + 标准投影），Dock 里不再比别的应用大一圈
- **UI 细节**：按设计稿逐屏比对的保真度修复（详见 commit）

## 测试

引擎单测 30 → 34（排序 / 类型推断与布局解耦 / 前缀收编 / frontmatter 显式类型）。
