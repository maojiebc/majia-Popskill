# v2.12.0 — 界面双语：简体中文 + English

Popskill 现在会说英文了。界面跟随系统语言自动切换：简体中文用户看到的和过去一字不差，英文（及其它语言）用户看到完整的英文界面——从英文 README 下载的朋友不再对着满屏中文猜按钮。

## 本地化

全部界面文案完成本地化：主屏、健康横幅、添加/设置/定时任务弹层、修复弹层、详情 peek、菜单栏、确认弹窗、toast 与错误信息（约 295 条，含 30 组英文单复数变形）。相对时间等日期格式也跟随界面语言，英文界面不再蹦出中文日期。系统语言不受支持时统一显示英文。

在「系统设置 → 通用 → 语言与地区 → 应用程序」里可以单独为 Popskill 指定语言。

## 工程

本地化权威源为单一 String Catalog（`swift-app/l10n/Localizable.xcstrings`），预编译进资源包；本地 CI 新增双向覆盖校验——代码里每条文案必须在目录中有词条，目录里每个词条必须被代码引用，漏一条都过不了检查。

刻意保持中文的部分：内置精选目录的 skill 简介（内容数据）、开发者日志。
