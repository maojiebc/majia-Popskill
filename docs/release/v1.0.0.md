# Popskill v1.0.0 — First public release

Popskill v1.0.0 is the first signed + notarized release. The app ships as a single
DMG; updates roll out through Sparkle.

## What's inside

**One matrix to see every AI capability.** The main view lines up every skill,
agent, CLI, MCP, and config against Claude Code and Codex. One click toggles a
skill on or off for each AI tool — no more juggling `~/.claude/skills/` vs
`~/.codex/skills/` by hand. The right-pane Inspector shows the SSOT path and
each app's symlink status so you can spot a broken link before your agent
starts misbehaving.

**⌘K command palette.** Search every capability, run a quick action (refresh
data, open Link Health, jump to Settings), or toggle Claude/Codex directly on
a result with ⌘1 / ⌘2.

**Token usage insights.** Popskill scans `~/.claude/projects/**/*.jsonl` in the
background and surfaces input/output/cache-create/cache-read totals plus the top
10 capabilities by consumption.

**Capability source library.** Add a GitHub catalog repo (e.g.
`anthropics/skills`) and Popskill keeps it tracked, flagging updates as they
land upstream.

**Sync across machines.** iCloud Drive sync is built in — push your skills root
to iCloud from one Mac and pull from another. Git remote is also supported for
folks who prefer a self-hosted setup.

**Safe-by-default uninstalls.** Every uninstall creates a backup snapshot in
`~/.popskill/backups`. Idle (60+ days, no apps enabled) capabilities surface in
their own view so you can decide: keep, turn into a stub, or uninstall.

**5-step onboarding wizard.** First launch detects which AI tools are installed
on the Mac, scans for existing capabilities, and helps you pick a sync provider.

## Requirements

- macOS 14 (Sonoma) or newer
- Optional: `git` for the GitHub source workflow + Git sync provider

## Install

1. Download `Popskill-1.0.0.dmg` from the Releases page.
2. Open the DMG and drag Popskill.app to /Applications.
3. First launch: macOS will verify the notarization stamp, then walk you
   through the onboarding wizard.

## Auto-update

Popskill includes Sparkle and checks for updates on launch. You can also
trigger a manual check from the menu bar (⌘U).
