# Popskill v1.0.1 — Sparkle auto-update

Small follow-up to [v1.0.0](./v1.0.0.md). Same feature surface; what's new is the
**auto-update wiring**.

## What changed

- `Info.plist` now carries `SUFeedURL` →
  `https://maojiebc.github.io/majia-Popskill/appcast.xml` and `SUPublicEDKey`
  → the EdDSA public key paired with the private key sitting in majia's
  Keychain.
- Every launch (and once a day after that) Sparkle fetches the appcast and,
  if a newer version is published, offers a one-click in-app upgrade.
- Each DMG carries an EdDSA signature in the appcast `enclosure` so a
  swapped binary will be rejected on the client.

## Migration from v1.0.0

v1.0.0 doesn't ship Sparkle config, so the first hop is manual: download
v1.0.1 from this Releases page and replace the app in /Applications. From
v1.0.1 onward Sparkle takes over.

## Verifying

```bash
# Confirm Sparkle config is baked into the Info.plist
/usr/libexec/PlistBuddy -c "Print :SUFeedURL :SUPublicEDKey" \
  /Applications/Popskill.app/Contents/Info.plist
```

Expected output includes:
- `SUFeedURL = https://maojiebc.github.io/majia-Popskill/appcast.xml`
- `SUPublicEDKey = h7HOqj21MlKe5UJFFa9GKBmV6MtdlcDSeJa9rmAguq8=`
