# Popskill v2.2.0 — 键盘导航 + 色彩语义定稿（设计 PATCH-02）

设计侧迭代定稿的五项优化，全部落地。

## 新增

- **键盘导航**：`↑↓` 行焦点（蓝环 + 随动滚动）、`←→` 选工具列、`空格/回车` 切换挂载或打开修复弹层、套装行空格折叠/展开、`Esc` 清焦点。过滤行右侧的提示文案换成键位说明。

## 变更

- **状态色语义**：「已激活」从电光蓝改为**绿色**（与同步指示统一）——pill、子项 ●、聚合分数、覆盖条、详情 peek 全套；电光蓝收窄为纯交互色（聚焦、焦点环、链接、新装高亮）。
- **首屏密度**：套装默认全部折叠，独立能力卡直接上首屏；点头部或按空格展开。
- **套装头部**：补充总 tokens 显示；聚合分数 / 版本 / 操作三列改固定宽度，与子项清单列严格对齐。
- **细节**：子项描述截断修正；版本+更新徽标不再折行。

升级即生效，无数据迁移。
