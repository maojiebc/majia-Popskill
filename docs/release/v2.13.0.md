# v2.13.0 — 设计稿落地：统计条 · 双视图 · 断链整卡红

这一版把 claude.ai/design 交接稿里「账本版」最终态的三块视觉/交互补齐，皮肤不变（还是熟悉的米白账本风）。

## 顶部统计条

主屏顶部新增一条统计：Skill / Agent / MCP / CLI / Bundle 各类型数量，加上 Claude Code 与 Codex CLI 各自的「已激活 / 未挂载」拆分——一眼看清装了什么、挂载到哪。

## 卡片 / 表格双视图

右上角新增切换：卡片矩阵看质感，账本表格看密度。表格一屏列出名称 / 类型 / 作者 / Claude / Codex / 版本 / Tokens 七列，套装是可展开的表头行（带覆盖分数与迷你进度条），子项树形缩进、状态用 ●—◐✕ 符号矩阵。两个视图共享同一份数据、过滤和键盘导航——↑↓ 选行、←→ 选工具列、空格切换。

## 断链整卡红

某个能力的链接断了，整张卡片现在会变红：红边、淡红底、红头像、`✕ 断链` 徽标、暗红描述——比过去只标一个红符号醒目得多，表格视图里对应的行也会整行变红。

## 新手体验打磨

这一版还做了一轮「假装自己是第一次下载的普通人」的体检，把会让新手卡住的地方补齐：

- **没装 git 的新 Mac**：安装 GitHub 来源的技能会先检测 git，没装就给一句「在终端运行 `xcode-select --install`」的人话提示，而不是弹一个看不懂的系统框；各类失败（私有仓库 / 找不到 / 断网）也都翻成了人话。
- **只用一个工具的人**：没装的工具（比如没装 Codex）现在标「未安装」并淡显，挂载前会先问一句——不再悄悄在你电脑上建出 `~/.codex`。
- **右键菜单**：卡片/表格行右键即可「在访达中显示 / 移除 / 查看详情」，核心动作不再只藏在悬停里。
- **状态图例**：过滤行常驻一行 `● 已激活 — 未挂载 ◐ 占位 ✕ 断链`，矩阵里的符号不再是密码表。
- **空态说人话**：第一次打开会解释「Popskill 帮你把技能装一次、同时挂给 Claude Code 和 Codex」。
- 一堆小修：导入本地技能前先确认并列清单、统计数字对比度提升、安装失败自动回滚不留残目录、文档措辞与备份份数订正。

## 工程

三块视觉 + 一轮新手体检共修 6 条上线拦路石 + 约 20 条细节，全部在现有账本皮肤上做加法，复用状态格、覆盖分数、详情 peek 等现成组件；中英双语同步、关键路径有单测覆盖，全套测试 93 项通过。
