# Popskill v2.4.0 — 目录来源提示：观远五件套归拢

## 新增

- **精选目录第五级来源提示**：`guanskill install-skill` 这类**复制安装**的 skill（不在 lock、无 .git、无 homepage——四级溯源全落空的「来源孤儿」），由内置目录补来源标注。观远五件套（guancli / guands / guanetl / guanvis / guanwf）现在归拢为一张 `guanskill` 套装卡。
- **npm 源套装**：归拢条件从仅 GitHub 放宽到 GitHub + npm（与设计原型一致）；npm 源的更新检查自动跳过——guanskill 的更新仍走 `npm update -g` + `guanskill install-skill`，诚实不装样子。
