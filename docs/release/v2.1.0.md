# Popskill v2.1.0 — 首个正式对外版本

本地 AI 能力管理器：**技能装一次，挂到多个 AI 工具上；坏了能修，旧了能升。**

store 在 `~/.agents/`（正在形成的跨工具约定目录），Claude Code / Codex CLI 侧只是 symlink——文件系统就是数据库，无 sidecar、无 SQLite。一个屏幕（能力矩阵）+ 添加/设置弹层 + 行内修复 + 详情 peek。

## 本版新增（相对 v2.0.0）

- **更新检查与一键更新**：对源目录算 SHA-256 内容哈希、拉上游比对——不依赖版本号，任何内容变化都能发现。↑ 徽标 / 「全部更新」/ 修复弹层「更新并修复」全部接通；启动后台自动检查，开了「自动更新」的源直接更新。
- **更新前自动备份**：旧版进 store 回收站（`~/.agents/.trash/`，保留 20 份），随时可恢复。
- **未托管目录导入**：设置 → Store「导入未托管目录」，一键把散落在 `~/.claude` / `~/.codex` 里的真实技能目录收编进 store 并换成 symlink——存量用户的迁移按钮。
- **安装路径安全校验**：拒绝路径遍历类目录名。

## 功能总览

- 能力矩阵：独立能力双列卡 + 套装通栏卡（整套挂载或逐子项开关，自动物化）
- 添加：粘贴 GitHub 仓库 / 本地路径 → 安装计划（`ln -s` 预览）→ 安装并链接
- 修复：断链 / 本地副本行内处理，危险操作全部走回收站
- 详情 peek：点能力名称看 SKILL.md 摘要
- 搜索（`/`）/ 类型过滤 / 命中高亮

## 边界

- 仅 macOS 14+，Developer ID 签名 + 公证，Sparkle 应用内更新
- 工具支持 Claude Code 与 Codex CLI（按需求收敛，暂不做更多）
- npm 源暂不支持
