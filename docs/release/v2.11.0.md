# Popskill v2.11.0 — 定时任务人性化重做 + 卡片信息密度

## 定时任务面板重做

v2.9 首版面板的主角是 plist 字段（reverse-DNS 标识符 + 完整命令行），看懂要靠脑补。这版把主角换成人关心的三件事：这是干嘛的、下次什么时候跑、上次跑成没。

- **分组按行为**：按时间跑 / 常驻后台 / 登录自启——launchd 还是 crontab 降级成名字旁的小标签。
- **下次运行是主信息**：「17 分钟后 · 23:41」蓝色倒计时，组内按它排序；StartCalendarInterval 与 cron 五字段（含 `*/n`、范围、列表、dom/dow OR 语义）精确计算。
- **上次结果带时间**：从日志文件修改时间推断，「上次 今天 05:00 ✓」；失败显示退出码。
- **人话名**：reverse-DNS 标识符自动美化，hover ✎ 可加自定义备注（存 Popskill 自己的 meta，不动 plist）。
- **停摆检测**：常驻 daemon 在册但进程没了 → 红条 + 上次退出码 + 行内「重启」。
- 顶部「今天还会跑 N 个」一句话摘要。

## 激活状态 pill 压缩

- 卡片右侧的工具 pill 不再整列撑满：常态只留「圆点 + 工具名」，自适应内容宽、整列右对齐——省出来的宽度全部还给左侧的名称和描述。
- 状态语义不丢：圆点配色四态不变，hover 提示带具体状态；**占位 / 断链警示态仍显示文字**，问题不会被藏起来。
