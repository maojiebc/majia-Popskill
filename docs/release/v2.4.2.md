# Popskill v2.4.2 — 异机套装归拢修复

第二台 Mac 实测反馈的修复版。

## 修复

- **同仓技能在某些机器上归拢不成套装**：当 skill 元数据里的来源是「仓库内深路径」（如 `github.com/owner/repo/tree/main/skills/x`，逐 skill 不同）时，归一化没有截到 `owner/repo`，导致每个技能各成一组、全部散开。现在 GitHub 系 URL 一律收敛到仓库根（支持 /tree/ /blob/ 深路径、www. 前缀、raw.githubusercontent.com、#锚点、?查询串、.git、SSH 形式）。
- **整 store 软链布局归拢不了**：dotfiles 同步类布局下 store 条目本身是 symlink，此前被一刀切排除在套装归拢外。现在照常归拢（显示层）；更新检查与执行仍逐成员跳过软链，安全语义不变，有专项单测锁死。
- 精选目录补充 `agent-reach` 简介。

测试 41 → 44（深路径归一化 7 变体 / 深路径 homepage 归拢 / 软链归拢 + 更新跳过）。
