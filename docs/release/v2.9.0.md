# Popskill v2.9.0 — 定时任务面板

缘起：majia-guanyuan 被本机 launchd 任务用旧缓存回滚（2026-06-12 事故），排查时发现「本机有哪些定时任务、什么时候跑、上次结果如何」只能靠 launchctl/crontab 命令，没有可视化入口。这版把它做进 Popskill。

## 定时任务面板（◷ / ⌘J）

- **launchd 用户级任务**：解析 `~/Library/LaunchAgents/*.plist` 的 Label / ProgramArguments / 调度配置，翻译成人话——StartCalendarInterval（"每天 06:00" / "每周一 10:17" / 多时间点）、StartInterval（"每 2 小时"）、KeepAlive（"常驻 daemon"）、RunAtLoad（"登录自启"）。
- **crontab 条目**：`crontab -l` 全部条目同屏展示，5 字段翻译（"每 6 小时的第 17 分"），命令与日志重定向自动拆分；只看不动，管理仍走 `crontab -e`。
- **状态与上次结果**：与 `launchctl list` 比对——绿点 = 已加载且上次退出 0，红点 + 「上次退出 N」徽章 = 上次失败，灰点 = 已停用。
- **操作**（全部先弹确认）：立刻跑一次（`launchctl kickstart -k`）、停用 / 启用（`unload` / `load`，plist 文件原样保留）；有 StandardOutPath 的任务一键打开日志（日志按天滚动不在时打开所在目录）。
- **降噪**：com.apple / Google / Clash 等系统与第三方任务默认隐藏，底栏开关一键展开。
- 边界：plist 内容只读，绝不改写；写操作仅上述三个 launchctl 动作。
